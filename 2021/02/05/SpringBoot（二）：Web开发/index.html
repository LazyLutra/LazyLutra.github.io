@RestController 注解

在 SpringBoot 中，接口返回数据非常简单，只需要在 Controller 上添加 @RestController 注解即可，这也是一个 SpringBoot 新增的注解。

在 @RestController 注解源码中可以看到，使用了 @Controller 和 @ResponseBody 注解，

而 @ResponseBody 可以将返回的数据转为 JSON 格式。

所以，在默认情况下，使用 @RestController 就可以返回 JSON 格式的数据了。
