Mybatis（一）：XML常用标签

1. resultMap

映射文件中的 <resultMap> 元素表示 SQL 返回结果的映射集，主要用来定义映射规则、级联的更新和定义类型转化器等。

1.1 主要结构

  • resultMap元素
    • id 属性是 resultMap 的唯一标识
    • type 属性表示需要的 POJO
  • 子元素
    • id：表示哪列是主键
    • constructor：如果 POJO 没有配置构造方法时，使用该属性进行配置
    • result：数据表普通列和 POJO 的映射关系
    • association、collection、discriminator：这三个属性用在级联情况下

属性名称
描述

resultSetType 这是针对 JDBC 的 ResultSet 接口而言，其值可设置为 FORWARD_ONLY（只允许向前访问）、SCROLL_SENSITIVE（双向滚动，但不及时更新）、SCROLLJNSENSITIVE（双向滚动，及时更新）

3. INSERT

映射文件中的 <inseri> 标签用于插入语句，执行完后将返回一个整数，表示该语句的影响行数。

大部分属性与 <select> 标签的基本属性相同，多了一些特有属性。

3.1 属性

属性名称
描述
keyProperty 将插入或更新操作时的返回值赋给 PO 类的某个属性，通常会设置为主键对应的属性。如果是联合主键，可以将多个值用逗号隔开。
keyColumn 设置第几列是主键，当主键列不是表中的第 1 列时需要设置。如果是联合主键，可以将多个值用逗号隔开。
useGeneratedKeys 该属性将使 MyBatis 使用 JDBC 的 getGeneratedKeys（）方法获取由数据库内部产生的主键，例如 MySQL、SQL Server 等自动递增的字段，其默认值为 false。
