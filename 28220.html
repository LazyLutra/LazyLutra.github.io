Ubuntu - 安装Docker

  1. 更新 APT，安装需要的 HTTPS 软件包和需要的证书
  1. 添加 GPG 秘钥
  1. 向 sources.list 中添加 Docker 软件源
  1. 更新 apt，安装 Docker
  1. 启动 Docker
  1. 当前不是 root 用户，需要建立 docker 用户组并加入
  1. 验证 Docker 是否安装成功
