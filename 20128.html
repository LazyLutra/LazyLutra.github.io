数据类型

数据类型总结
类型 存储需求 取值范围
整形 int 4字节 -2,147,487,648 ~ 2,147,487,647（正好超过20亿）
short 2字节 -32,768 ~ 32,767
long 8字节 -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
byte 1字节 -128 ~ 127
浮点型 float 4字节 有效位数 6 ~ 7 位
double 8字节 有效位数 15 位
char char 与编码格式有关，unicode为 2 字节 \u0000（十进制等效值为 0） ~ 最大值是 \uffff（即为 65535）
boolean boolean 没有规定大小，单个 boolean 占 1 字节，数组 boolean[] 中每个元素占 4 字节[^1] 只有 true 和 false

对象与类

对象

所有的 Java 对象都是在堆中构造的。

  • 对象的行为（behavior)
    可以对对象施加哪些操作，或可以对对象施加哪些方法？
  • 对象的状态（state )
    当施加那些方法时，对象如何响应？
  • 对象标识（identity )
    如何辨别具有相同行为与状态的不同对象

不要返回引用可变对象的方法

可变对象：对象被创建后，状态可以被改变
不可变对象：对象被创建后，状态不可变

类

类之间的关系

  • 依赖（uses-a）
  • 聚合（has-a）
    例如，一个 Order 对象包含了 Item 对象
  • 继承（is-a）

构造器

  • 构造器与类同名
  • 每个类可以有一个或以上的构造器
  • 构造器可以有 0 个、1 个或者多个参数
  • 没有返回值
  • 构造器总是伴随 new 操作一起使用

方法参数

Java 总是使用按值调用，方法得到的是参数值的拷贝，并且不能修改传递给方法的参数变量内容。
对象引用是按值传递的。

方法参数具有两种类型：

  1. 基本数据类型
  2. 对象类型

方法参数使用情况：

  1. 一个方法不能修改基本数据类型的参数（数值型或者布尔型）
  2. 一个方法可以改变对象参数的状态
  3. 一个方法不能让对象参数引用一个新的对象
